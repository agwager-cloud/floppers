FOSTER'S FLOPPERS — HOTFIX 08

This update changes only the Start Scene overlay.

Changes:
- Replaces the two large side pods with one very thin control strip.
- Positions the strip in the clear arena-light band directly below the title.
- Keeps all referee and player faces and bodies unobstructed.
- Removes the extra "Enter the Tournament" heading because the main title already identifies the game.
- Keeps the name field, room-code field, Host button and Join button anchored to the Phaser canvas.

Apply:
1. Stop the local game with Ctrl+C.
2. Extract the ZIP contents directly into C:\Projects\floppers.
3. Choose Replace All.
4. Run npm run dev.

No npm install or server update is required.
